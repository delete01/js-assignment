Various methods in console function as follow:

console.log()

console.log() is used to output a message to the console.

console.table()

Display tabular data as a table.

console.time()

Starts a timer(can track how long an operation takes).

console.timeEnd()

Stops a timer that war previously started by console.time().

console.trace()

Outputs a stack trace to the console.

console.warn()

Outputs a warn message in console.

console.clear()

Clears the console.

console.count()

Logs the number of times that this particular call to count() has been called.

console.error()

Outputs an error message to the console.

console.assert()

Writes an error message to the console if the assertion is false.

console.group()

Creates a new inline group in the console. This indents following console message by an additional level, until console.groupEnd() is called.

console.groupCollapsed()

Creates a new inline group in the console. However, the new group is created collapsed. The user will need to use the disclosure button to expand it.

console.groupEnd()

Exits the current inline group in the console.

These all are the methods of Console Object.
